
# Emailer Client

Essa aplicação é o client-side de um desáfio prático desenvolvido pra um processo seletivo.

Este projeto requer o **server-side** para funcionar. Certifique-se de ter o backend rodando antes de iniciar o client.

**Repositório do server-side:** https://github.com/victorgfz/emailer_server.git

## Instalação

```bash
# Clone o repositório
git clone https://github.com/victorgfz/emailer_client.git
cd emailer_client

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`

## Tecnologias utilizadas

- Next.js
- TypeScript
- Tailwind CSS
